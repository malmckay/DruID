DruID searches for dru repeats.

DruID identifies anchor sequences flanking the dru repeats and then idenitifies each of the repeats in order. The repeat profile is used to identify the dru type (dt).

If unknown repeats are listed, a dru type is not identified. In this case, the trace file for that sequence should be sent to the curator for annotation (info@dru-typing.org). The trace file should also be sent in the case of a new combination of known repeats, as this constitutes a new dru type.

DruID also identifies truncated hypervariable regions, which are annotated as dt0.

Regular Updates
This application was last updated on June 1st, 2011. There are currently 75 dru repeats and 314 dru types in it's local database.
