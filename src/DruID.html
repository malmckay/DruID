Drop your sequence files, folders or text, onto this window.
Sequence files in fasta format can be dragged individually, in groups or in folders onto this window. The dru repeat profiles will be calculated and appear automatically.
Reverse Complementing not necessary.
There is no need to reverse complement your sequence files, DruID will automatically check the reverse complement, and display results if they are found for the reversed complemented sequence.
DruID searches for dru repeats.

DruID identifies anchor sequences flanking the dru repeats and then idenitifies each of the repeats in order. The repeat profile is used to identify the dru type (dt).

If unknown repeats are listed, a dru type is not identified. In this case, the trace file for that sequence should be sent to the curator for annotation (info@dru-typing.org). The trace file should also be sent in the case of a new combination of known repeats, as this constitutes a new dru type.

DruID also identifies truncated hypervariable regions, which are annotated as dt0.

Copy and Paste
Copy and paste is not supported at this time. Please use the "Save Results As..." button to export the results. This will save a file in CSV format, which can be opened in Microsoft Excel.
