Drop your sequence files, folders or text, onto this window.
Sequence files in fasta format can be dragged individually, in groups or in folders onto this window. The spa repeat profiles will be calculated and appear automatically.
Reverse Complementing not necessary.
There is no need to reverse complement your sequence files, SpaID will automatically check the reverse complement, and display results if they are found for the reversed complemented sequence.
SpaID searches for spa repeats.
Copy and Paste
Copy and paste is not supported at this time. Please use the "Save Results As..." button to export the results. This will save a file in CSV format, which can be opened in Microsoft Excel.
Regular Updates
This application was last updated on May 26th, 2012. There are currently 538 spa repeats and 10491 spa types in it's local database.
